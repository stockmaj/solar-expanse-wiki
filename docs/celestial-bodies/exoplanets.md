# Exoplanet Systems

Four star systems reachable only through interstellar travel, via a
generation ship constructed in the late game. Each section below lists
every body the game generates in the destination system, with its mass,
radius, and orbital elements around the host star.

_Semi-major axis here is measured around the host star, not the Sun._
