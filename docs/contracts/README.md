# Contracts

Contracts drive progression in Solar Expanse — they're the game's source of
funding alongside resource sales. Each contract is a set of objectives — usually
"build facility X on body Y" or "deliver Z to a destination" — that pay
out cash, resources, or unlocks when complete. Many contracts also unlock
the next link in a chain (Mars Phase 1 → Mars Phase 2 → …), a new spacecraft,
or a new launch vehicle.

## Reading the table

- Rows are sorted by **Order** — the contract's dependency depth in the unlock DAG. Order 0 contracts are roots with no prerequisite; Order 1 is unlocked by an Order 0 contract; and so on. Ties break alphabetically.
- A contract marked **(final)** ends a campaign chain.
- **Requirements**: the objectives you have to complete to claim the payout. Body-specific objectives ("deliver 100 t to Mars") list the *what* but not the destination — the premise text describes the target.
- **Rewards**: cash, resources, facility / spacecraft / launch-vehicle unlocks, and the next contract in the chain.
- **Premise**: the in-game flavor text introducing the contract.
