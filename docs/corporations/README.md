# Corporations

The five playable starting factions in Solar Expanse. Each scenario
(Colonization Era / The Expansion / Race Beyond) ships a different
pre-built save where every corporation starts with its own completed
research, funding, and fleet. Difficulty further scales starting money
and ongoing costs.

## Difficulty

Selecting a difficulty on the New Game screen scales the corporation's pre-built starting state before play begins, and modifies upkeep / supply usage for the rest of the run.

| Difficulty | Starting money | Upkeep cost | Supply usage |
| --- | --- | --- | --- |
| **Explorer** | ×1.25 | ×0.5 | ×0.5 |
| **Pioneer** | ×1 | ×1 | ×1 |
| **Veteran** | ×0.75 | ×1.5 | ×1.5 |
